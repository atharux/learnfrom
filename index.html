How to use this course

Three ways people work through it — pick the one that fits your situation.

01

Weekend sprint

Read the full course in a weekend (4–6 hours). Use it as a complete mental model before starting a new project or redesign. Skip the exercises and come back to them when you have a real problem to apply them to.

02

Module by module with a live project

Work through one module per week alongside an active project. Do the exercises against real users and real data. By module 8 you have a complete DMAIC case study you can show.

03

Reference and toolkit

Keep it open on your desk. Use the heuristic worksheet before a design review, pull the persona template when starting research, run the DMAIC starter when scoping an improvement project. It works as a reference for the frameworks you'll use repeatedly.

DMAIC applied to UX design. Define, measure, and prove the improvement from your design work using real process data. Full warehouse dashboard redesign case study included.
